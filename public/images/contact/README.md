# Imágenes - Contacto e Información

Esta carpeta contiene imágenes para la página de contacto e información.

## Imágenes sugeridas:

### Ubicación:
- `map-location.jpg` - Mapa o vista aérea
- `neighborhood.jpg` - Vecindario cercano
- `street-view.jpg` - Vista desde la calle

### Recepción:
- `reception-desk.jpg` - Escritorio de recepción
- `front-desk-staff.jpg` - Personal de recepción
- `lobby-contact.jpg` - Lobby área

### Contacto:
- `phone-line.jpg` - Línea telefónica
- `email-contact.jpg` - Correo de contacto

### Ubicaciones (sucursales):
- `paris-property.jpg` - Propiedad París
- `mexico-city-property.jpg` - Propiedad Ciudad de México

## Especificaciones:

- **Ancho**: 800px - 1200px
- **Altura**: 600px - 1000px
- **Formato**: JPG/JPEG optimizado
- **Calidad**: 75-85%

## Cantidad recomendada:

- **Total estimado**: 8-12 imágenes

## Notas:

- Profesionalismo y accesibilidad
- Mostrar amabilidad del equipo
- Claridad de ubicación
- Transmitir confianza y disponibilidad
